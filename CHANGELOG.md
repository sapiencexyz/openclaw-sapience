# Changelog

## [1.0.0] - 2025-02-02

Initial release.

- GraphQL queries for markets, conditions, positions
- WebSocket taker flow (start auction, receive bids, accept)
- WebSocket maker flow (listen for auctions, submit bids)
- EIP-712 signing for maker bids
- Claiming flow (burn settled positions)
- Reference mm-listener.js script
- Heartbeat docs for both taker and maker roles
